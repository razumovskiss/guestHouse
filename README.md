# Hotel Management System

## Система управления номерами гостиницы на чистом JS.

### Что делает?

- Регистрация гостей
- Заселение в номера
- Отображение 18 номеров (стандарт/комфорт/люкс)
- Выселение

## Запуск

Открыть index.html локально в браузере, никаких зависимостей нет.

## Файлы

- index.html - код сайта
- style.css - стили

## Данные хранятся в LocalStorage ( кэш браузера ).
